React Native






React Native combines the best parts of native development with React, a best-in-class JavaScript library for building user interfaces and mobile applications.

Some Basic Components:

(Click on Component for example code)

View:

The most fundamental component for building a UI, View is a container that supports layout with flexbox, style, some touch handling, and accessibility controls. View maps directly to the native view equivalent on whatever platform React Native is running on, whether that is a UIView, div, android.view, etc.

Text:

A React component for displaying plain text. Text supports nesting, styling, and touch handling. All text have to be in Text Components.

StyleSheet:

A StyleSheet is an abstraction similar to CSS StyleSheets By moving styles away from the render function, it makes the code easier to understand. Naming the styles is a good way to add meaning to the low level components in the render function.
